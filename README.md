# React Tutorial - Tic Tac Toe

Sample app created from following the [Intro to React Tutorial][link-to-tutorial]

## Install Dependencies

```shell
npm install
```

## Run in Development Mode

```shell
npm start
```

This should open http://localhost:3000 in your browser.

## Formatting

```shell
make format
```
